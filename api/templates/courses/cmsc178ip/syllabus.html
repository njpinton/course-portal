Course Description

This course covers the fundamental concepts and techniques in digital image processing and computer vision. Topics include image representation, enhancement, restoration, segmentation, feature extraction, and an introduction to deep learning for computer vision applications.

Prerequisites

  • CMSC 21 - Fundamentals of Programming
  • CMSC 123 - Data Structures
  • Math 23 - Linear Algebra (recommended)

Learning Objectives

  1. Understand the mathematical foundations of digital image representation
  2. Apply image enhancement and filtering techniques
  3. Implement image restoration and geometric transformation algorithms
  4. Extract features and perform image segmentation
  5. Apply deep learning techniques to computer vision problems
  6. Develop practical skills using Python, OpenCV, and deep learning frameworks
