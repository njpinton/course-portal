Brain.js
Pure JavaScript

No dependencies, no WebGL required. Brain.js is perfect for simple neural networks that run anywhere JavaScript runs.

const brain = require('brain.js');
const net = new brain.NeuralNetwork();

net.train([
  { input: [0, 0], output: [0] },
  { input: [1, 0], output: [1] },
  { input: [0, 1], output: [1] },
  { input: [1, 1], output: [0] }
]);

const output = net.run([1, 0]); // [0.98]
  • Tiny Footprint — ~15KB minified
  • Node.js Ready — Server-side ML
